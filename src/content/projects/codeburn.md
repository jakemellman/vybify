Track token usage, cost, and performance across 18 AI coding tools. Breaks down spending by task type, model, tool, project, and provider. Runs locally — no wrapper, proxy, or API keys required. Reads session data directly from disk and prices every call using LiteLLM.

Install with `npm install -g codeburn` or `brew install getagentseal/codeburn/codeburn`, or run directly with `npx codeburn`.
